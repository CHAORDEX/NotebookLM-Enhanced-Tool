# NotebookLM 增强插件 (NotebookLM Enhanced Tool)

这是一个专为 Google NotebookLM 打造的 Chrome/Edge 浏览器增强插件，旨在提升用户在资源管理、思维导图处理和文件下载等场景下的使用体验。

## 🌟 核心功能

### 1. 🗂️ 资源批量管理 (Batch Resource Management)
- **批量删除**：在右侧资源列表中自动注入复选框，支持一键全选和批量删除多余资源，免去逐个点击菜单确认的繁琐操作。
- **批量下载**：支持一键批量下载用户勾选的多个资源。
- **操作面板**：悬浮提供直观的选中数量统计、处理进度显示和防误删确认机制。

### 2. 🧠 思维导图增强 (Mindmap Enhancements)
- **自动展开节点**：打开思维导图资源时，自动触发展开所有节点，无需手动点击。
- **复制为 Markdown**：一键将思维导图的树状结构解析并复制为标准的 Markdown 文本，方便直接粘贴到 Notion、Obsidian 等笔记软件中。
- **下载为 Markdown**：支持将思维导图直接导出并下载为独立的 `.md` 格式文件。

### 3. 💾 智能下载与重命名 (Smart Download Naming)
- **精准的智能重命名**：自动拦截 NotebookLM 的下载动作，根据来源文档名称或资源自身标题，对下载文件进行智能重命名，彻底告别无意义的 `unnamed` 或 `下载` 等默认文件名。
- **修复媒体扩展名**：修复原生系统在下载图片、音频、视频时可能出现的双后缀（如 `.md.png`）或后缀丢失问题，确保文件格式正确。

## 🚀 安装指南

1. 下载或克隆本项目的源代码，并解压。
2. 打开基于 Chromium 的浏览器（如 Chrome, Edge），在地址栏输入并进入扩展程序页面：`chrome://extensions/` （或 `edge://extensions/`）。
3. 开启页面右上角的 **开发者模式** (Developer mode)。
4. 点击左上角的 **加载已解压的扩展程序** (Load unpacked)，选择刚才解压的插件根目录（包含 `manifest.json` 的目录）。
5. 安装完成！打开或刷新 [NotebookLM 网页](https://notebooklm.google.com/) 即可体验增强功能。

## 🛠️ 隐私与安全

本插件仅在客户端（浏览器本地）运行，不会收集、存储或上传任何用户隐私数据和笔记内容。所有的下载拦截、重命名和批量操作均在浏览器本地的 DOM 层和请求拦截层完成。
